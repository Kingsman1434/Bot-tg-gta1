Telegram Bot for Game Logs Analysis (GTA RP) 🤖
Профессиональный инструмент для администраторов игровых серверов GTA RP, автоматизирующий мониторинг и анализ игровых логов. Бот позволяет оперативно выявлять нарушения правил сервера, махинации с валютой и подозрительные действия на аккаунтах.

🚀 Основные возможности
Скрапинг без API: Автоматизированный сбор данных с закрытого веб-ресурса логов.

Авторизация и сессии: Поддержка работы через Cookies и Session ID для доступа к защищенным разделам.

Умный парсинг: Извлечение структурированной информации из сырого HTML (игровые события, таймстампы, ID игроков).

Многопоточность: Обработка запросов в несколько потоков для ускорения получения данных.

Уведомления: Мгновенный вывод результатов анализа прямо в интерфейс Telegram.

🛠 Стек технологий
Language: Python 3.x

Libraries: * telebot (или aiogram) — взаимодействие с Telegram API.

Selenium WebDriver — эмуляция действий пользователя и навигация по сайту.

BeautifulSoup4 — парсинг и очистка данных.

Threading — параллельное выполнение задач.

Data Storage: JSON (конфигурация и временные данные).

📋 Архитектура решения
Auth Module: Управляет жизненным циклом сессии (логин, хранение кук).

Parser Engine: Находит нужные таблицы в DOM-дереве, обрабатывает динамические URL и пагинацию.

Bot Interface: Обрабатывает команды администраторов и форматирует вывод (использование Markdown для читаемости логов).
